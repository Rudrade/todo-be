# TodoBe
This is the api backend for the Todo project, made with Spring Boot.

## Todo Project
This is a simple project for task management, where a user with the admin profile can also management the app's users.<br>
The main goal of the project was to learn the following:
* Angular
* CI/CD on GitHub
* AWS Deployment
* Rate limiting in Spring using Bucket4j+Redis
* Spring Testcontainers

The project is divided by a
<a href="https://github.com/Rudrade/todo-be">Spring Boot backend </a>
and an 
<a href="https://github.com/Rudrade/todo-fe">Angular frontend<a/></b>
